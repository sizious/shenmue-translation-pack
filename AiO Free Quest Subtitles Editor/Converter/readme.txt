Shenmue II Free Quest Subtitles Editor v1.xx to v1.02+ Exporter
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

[ I. DESCRIPTION ]

This tool was made for retrieve and export subtitles from *MESSED UP* NPC 
"PAKS" files. In fact, public releases of the FQ Editor (down to v2.0) haves a 
very *BAD* hack algorithm. The FQ Editor v2.xx+ hacks NPC "PAKS" files
properly, but only from ORIGINAL, *NOT BAD HACKED* "PAKS" files.

[ II. COMPILING ]

Simply open the "convert.dproj" file and compile it.
