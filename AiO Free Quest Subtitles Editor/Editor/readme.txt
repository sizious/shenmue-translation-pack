Shenmue AiO Free Quest Subtitles Editor
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

[ I. DESCRIPTION ]

Shenmue AiO Free Quest Subtitles Editor is an Open Source software used to 
modify in-game NPC (Non-Playing Characters) subtitles text of Shenmue Series, 
including:

	- What's Shenmue (DC)
	- Shenmue I (DC)
	- Shenmue II (DC / XBOX)

You can talk to these NPC in-game in Free Quest mode (aka in the game's 
streets). If you want to translate cinematics subtitles, please use the
"Shenmue AiO Subtitles Editor" instead.

[ II. COMPILING ]

This app was developed in Delphi 2007.

To compile this app you'll need:

- Delphi 7 or superior (XML units needed)
- JCL/JVCL components: http://homepages.codegear.com/jedi/jvcl/
- DFF Lib: http://www.delphiforfun.org/Programs/Library/Default.htm
- DCL: http://sourceforge.net/projects/dclx/
