 __| Shenmue AiO Free Quest Subtitles Editor |_________________________________

  Shenmue AiO Free Quest Subtitles Editor is a free software used to modify 
  in-game NPC (Non-Playing Characters) subtitles text of Shenmue Series, 
  including:
  
    - What's Shenmue (DC)
    - Shenmue I (DC)
    - Shenmue II (DC / XBOX)
	
  You can talk to these NPC in-game in Free Quest mode (aka in the game's 
  streets). If you want to translate cinematics subtitles, please use the
  "Shenmue AiO Subtitles Editor" instead.
 __         ___________________________________________________________________
 __| USAGE |___________________________________________________________________

  NPC files are located in the AFS file called "HUMANS.AFS". This AFS is 
  located in 'SCENE\<DISC_NUM>\NPC' folder.
  
  First, you must unpack the "HUMANS.AFS" to a folder. Use the "AFS Utils" tool
  to do this. After that, you can use the "Select directory" from the main menu
  in the Free Quest editor to retrieve each NPC character to edit.
 __                 ___________________________________________________________
 __| SOFTWARE iNFO |___________________________________________________________

  This software use extra files. All these files are located in the "data"
  directory.
  
  Structure of the "data" directory:
    - npc_info.csv   : Contains NPC informations (Gender...).
    - chrlistX.csv   : This file is used for accentuated characters support. 
                       Open it with a text editor to see available characters.
                       Two versions are available, for Shenmue I and II.
    - faces          : This directory contains NPC faces images.
