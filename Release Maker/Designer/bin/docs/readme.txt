 __| Shenmue Release Maker |____________________________________________________

  The purpose of this application is to generate self-extracting packages with
  an original disc authentification code. You can use this to release your
  completed translation project!

  In clear, the self-extracting package will work only if the end-user puts
  the required CD-ROM or DVD-ROM in the drive. If not, the package is useless
  because it's encrypted with 3 strong algorithms (AES-256, PC1 and Camellia).
  
  The application is fully configurable (UI messages, pictures, icons, colors,
  and more).
 __         ____________________________________________________________________
 __| USAGE |____________________________________________________________________

  Just starts the "Release Maker" program and input at least:
    1. The "Source Directory" in the "Main" tab, to specify where the files to
       release are located in.
    2. The release information ("Author", "Game Name"...) for your distribution.
    3. The "EULA" information in the "EULA" tab. You should use a RTF file.
    4. In the "Config" tab, the "language UI" for the output package.
    5. One "media key" from your disc (you must insert the disc in your 
       drive).
    6. Then click "Make" to start making the package.
	
  It strongly recommanded to test the output package when the process is done,
  before releasing it.
