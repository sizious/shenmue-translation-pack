ENGLISH:

Shenmue AiO Subtitles Editor is an Open Source software used to modify the in-game subtitle text of Shenmue I, on Dreamcast, and Shenmue II, on Dreamcast and Xbox.

* NOTE: The file 'chars_list_one.csv' and 'chars_list_two.csv' are used for accentuated characters support. Open them with a text editor to see available characters.

--------------------

FRANÇAIS:

Shenmue AiO Subtitles Editor est un logiciel Open Source qui permet de modifier les textes des sous-titres de Shenmue I, sur Dreamcast, et Shenmue II, sur Dreamcast et Xbox.

* NOTE: Le fichier 'chars_list_one.csv' et 'chars_list_two.csv' sont utilisés pour le support des caractères accentués. Ouvrez-les avec un éditeur de texte pour voir les caractères disponibles.
