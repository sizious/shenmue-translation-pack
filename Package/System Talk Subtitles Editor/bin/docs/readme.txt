___| Shenmue System Talk Subtitles Editor |_____________________________________

  This program is an editor for some special files called SYSTALK. These files 
  contains some subtitles used in particular game events, by example when Ryo 
  should go home (you know, when Ryo stops to walk and says 'It's time to go 
  home', forcing you to save the game...). 
  
  This editor is only available on Console version yet (because making an UI 
  is very long to do, it requires a lot of time).
___         ____________________________________________________________________
___| USAGE |____________________________________________________________________

  Many files can be edited with this tool:
    - /MISC/FREETALK.BIN
    - /MISC/FREETLK1.BIN
    - /MISC/FREETLK2.BIN
    - /MISC/FREETLK3.BIN
    - /MISC/FREETLK4.BIN
    - SYSTALK.BIN
    - SYSTALK in /MISC/COLD.BIN (use IPAC Browser to edit this file)
    - SYSTALK in /MISC/COLD_X.BIN (where 'X' is the language letter, aka 'F')

  For your convenience, two batches files are included:
    - extract: Extract all SCNF sections from SYSTALK.BIN (rename if needed)
    - inject : Inject all dumped SCNF sections to SYSTALK.BIN (rename if needed)
