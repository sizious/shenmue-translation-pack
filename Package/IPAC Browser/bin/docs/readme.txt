 __| Shenmue IPAC Browser |_____________________________________________________

  Shenmue IPAC Browser is a free utility for PKS/PKF/BIN files, allowing you to
  explore the contents of the Shenmue data files. The IPAC section is a special
  section allowing to store datas, like a ZIP file.
  
  The supported games are:
    - What's Shenmue (DC)
    - Shenmue I (DC)
    - Shenmue II (DC / XBOX)
  
  This can be useful to replace some IPAC entries contained in the Shenmue
  data files.
 __         ____________________________________________________________________
 __| USAGE |____________________________________________________________________

  PKS/PKF/BIN files are located in various folders in this game. Only data 
  files containing a non-empty IPAC section can be opened in this software. The
  software supports drag and drop, the association with the Windows Explorer or
  passing the file to open by the command line.
  
  You can import/export datas from/to the Shenmue file package, view the 
  informations on these data files and much more.
