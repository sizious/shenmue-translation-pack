___| Shenmue AFS Utils |________________________________________________________

  This tool allows opening, creating and editing AFS files packages. AFS file 
  packages are archives like WinRAR ones but with no compression. Many games 
  are using this file format. The first interest of it is to gather many files 
  in single big one. In some special cases, an index file (IDX) is used with 
  the AFS archive.
___         ____________________________________________________________________
___| USAGE |____________________________________________________________________

  Each time you'll see an AFS file in the Shenmue disc, you can open it with
  this tool.
