                         /_/ I. DESCRiPTiON       
						 
  This tool pack was made to convert Yamaha AICA Stream files (.STR) to RIFF
  WAVE files (.WAV).

  The STR format is used in several Shenmue Dreamcast games. With this tool, 
  you can listen to STR files. This stuff can help you for the translation.

  Enjoy this !

      II. USAGE /___/      						 
	
  To use this, you'll need the Dreamcast .STR datas of the episode you want to 
  translate. Copy them in a directory and the tool pack, then run the command:
  
    str2wav <infile.str> <outfile.wav>
    str2wav <infile> (where <infile> is <infile.str> without extension)

  For examples you can enter the following commands:
  
    str2wav A0114A001
  This will generate the "A0114A001.wav" file from "A0114A001.str".

    str2wav wazamakimono.str myfile.wav
  This will decode the "wazamakimono.str" file to the "myfile.wav".

  A batch file was integrated. You can copy your .STR files to the ".\Input\"
  directory, double-click on the "batch.cmd" file then retrieve your converted
  files in the ".\Output\" directory.

  That's all folks.
