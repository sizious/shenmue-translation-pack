 __                             ________________________________________________
 __| Shenmue Sections Explorer |________________________________________________

  This very simple editor was made to view/edit the content of every Shenmue 
  data files. It works with all Shenmue versions.

  This tool will analyze the structure of the data files and allow you patch 
  the sections you need to. Remember that you must use the "IPAC Browser"
  program to edit the content of "IPAC" sections.
 __         ____________________________________________________________________
 __| USAGE |____________________________________________________________________

  The usage is exactly the same as "IPAC Browser". Just run the tool, select
  a Shenmue data file (for example a "MAPINFO.BIN" file, a ".PKS"...), then go.
  
  You can import/export datas from/to the Shenmue file package, view the 
  informations on these data files and much more.
