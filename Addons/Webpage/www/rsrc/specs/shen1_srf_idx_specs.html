Feuille 1: SRF
Length Type Description
En-tête
4 LongInt Longueur du nom du personnage, habituellement 8
4 String Nom du personnage
Sous-titre
4 LongInt Longueur du sous-titre Si longueur = 4, pas de texte
n String Texte du sous-titres
n Byte Bytes nuls (zéro)
Données
4 LongInt Taille des données (en bytes)
n Byte Données
n Byte Bytes nuls (zéro), si nécessaire Bloc de 2048 bytes

Feuille 2: IDX
Length Type Description
En-tête
4 String « IDX »
1 Byte « 0 » (zéro)
2 ShortInt Nombre de fichiers – total
2 ShortInt Nombre de fichiers – moins les SRF
12 Byte « 0 » (zéro)
Pour chaque fichiers qui n'est pas un SRF
10 String Nom du fichier (sans extension)
2 Byte « 0 » (zéro)
2 ShortInt Numéro du fichier (moins 1)
2 ShortInt Numéro de fichier du SRF lié (moins 1)
4 LongInt Offset du sous-titre dans le SRF
