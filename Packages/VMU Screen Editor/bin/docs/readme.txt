 __| Shenmue VMU Screen Editor |________________________________________________

  This editor is a extra bonus for you, Shenmue fans! It allow you to edit the
  LCD screen of the VMU, in order to translate some actions (for example 
  "Saving"...).
  
  The supported games are:
    - Shenmue I (DC)
    - Shenmue II (DC)
  
  This utility is only for the Dreamcast, because the VMU is a Dreamcast 
  peripheral only. What's Shenmue isn't supported because AM2 didn't use the 
  LCD VMU in this game.
 __         ____________________________________________________________________
 __| USAGE |____________________________________________________________________

  You must have IPAC Browser on your disk before proceeding, because we must
  extract the LCD.IWD file from the MISC/COLD_*.BIN file. To do that:

   1. Run IPAC Browser
   2. Select the right COLD_*.BIN file (COLD.BIN, COLD_E.BIN, COLD_F.BIN...) in
      the MISC folder for Shenmue I, or the MISC/COLD.BIN for Shenmue II and
      open it in IPAC Browser.
   3. In the list, right-click on the LCD.IWD "LCD Table" and select the
      "Export..." option. Save the file in your prefered location.
   4. Open the "LCD.IWD" file in Shenmue VMU Screen Editor.
   5. Have fun! This tool is similar to use as IPAC Browser.

  When you done, import the modified LCD.IWD with IPAC Browser too, using the
  reverse procedure (I know, YOU know how to do that!).

  Please note you should replace a LCD data with a bitmap identical in size,
  but if you want patch a LCD entry with a smaller or bigger bitmap entry. I
  don't know what is the impact in the game when you do that.
