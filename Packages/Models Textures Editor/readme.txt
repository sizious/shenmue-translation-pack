                         /_/ I. DESCRiPTiON          
					  
Shenmue Models Textures Editor is an Open Source software used to 
modify the models files MT5/MT6/MT7 of Shenmue Series, including:

	- What's Shenmue (DC)
	- Shenmue I (DC)
	- Shenmue II (DC / XBOX)

This can be useful to translate some specials textures (in-game objects like
piece of papers, etc.).

   II. COMPiLiNG /_/              /___/  

This app was developed in Delphi 2007.

To compile this app you'll need:

- Delphi 7 or superior (XML units needed)
- JCL/JVCL components: http://homepages.codegear.com/jedi/jvcl/
