ENGLISH:

SPR Utils is an Open Source software used to extract and create SPR files, composed of graphics files. These files are used in some video games, like Shenmue 1 or Shenmue 2.

--------------------

FRANÇAIS:

SPR Utils est un logiciel Open Source qui permet d'extraire et de créer des fichiers SPR, contenant des éléments graphiques, présents dans certains jeux vidéo comme Shenmue 1 ou Shenmue 2.
