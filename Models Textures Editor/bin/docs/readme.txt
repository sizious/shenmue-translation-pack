 __| Shenmue Models Textures Editor |___________________________________________

  Shenmue Models Textures Editor is a free editor for MT5/MT6/MT7 files, used 
  in case of some sprites must be translated. MTx are files containing object 
  models and textures.
  
  The supported games are:
    - What's Shenmue (DC)
    - Shenmue I (DC)
    - Shenmue II (DC / XBOX)
  
  This can be useful to translate some specials textures used in objects, like 
  objects containing a piece of paper telling you where to go in the game. In 
  this case, the texture contains some (english) text drawed on it.
 __         ____________________________________________________________________
 __| USAGE |____________________________________________________________________

  MTx files are located in various directories. The most common is the 
  "\MODEL\" directory located in the game CD root. Only MTx files containing 
  valid textures section can be edited in this software.
  
  You can import/export textures from/to the MTx package. Note that you'll need
  an external graphic editor like Adobe Photoshop. The used texture formats are 
  Dreamcast PVR and Microsoft Xbox DDS (DirectX). For this point, you can
  download some additional tools from the SHENTRAD website to help you on this.
